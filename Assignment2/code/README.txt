
README
***********

1. The cans locations are specified in the file cans.txt. You can change (add / erase) cans on it.
2. The main file is assignment2.cpp while the algorithm is implemented on pickupGuy.cpp
3. To call the algorithm: 
		 pg.test_EGreedy(5, 200, 0.01);
		where 5 is the number of episodes, 200 the number of plays and 0.05 is epsilon. You may change
		this values

5. Run the CMake the usual way
6. Call the executable with the cans.txt file as argument, ie.: "./assignment2 cans.txt"
